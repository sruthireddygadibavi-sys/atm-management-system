# ATM Management System

A console-based ATM Management System developed using Core Java. This project simulates basic ATM operations such as PIN verification, balance inquiry, cash deposit, cash withdrawal, and account security using a menu-driven interface.

## Features

- Secure PIN Authentication
- Balance Inquiry
- Cash Deposit
- Cash Withdrawal
- Limited Login Attempts for Security
- User-Friendly Menu-Driven Interface

## Technologies Used

- Java
- Object-Oriented Programming (OOP)
- Scanner Class for User Input

## How to Run

1. Compile the program:

```bash
javac ATM.java
```

2. Run the program:

```bash
java ATM
```
